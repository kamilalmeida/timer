## 💻 Features do projeto
- Responsividade

## ✔️ Técnicas e tecnologias utilizadas

- React Hooks,
- React Router,
- Styled-components
- Context API


## Abrir e rodar o projeto:
- Após baixar o projeto, você pode abrir em uma IDE de sua preferência.

- Instalar as dependências:
### `npm install`

- Para abrir o projeto no navegador:
### `npm run dev`
